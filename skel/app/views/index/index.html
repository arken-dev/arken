Enquanto pessoas procuram a solução de seus problemas em novas linguagens de
programação, a verdadeira revolução se dá com aquilo que já temos em mãos.

Linguagem c++ e lua se complementam, e com empenho e dedição uma nova estrutura
pode ser montada juntando aquilo que definitivamente queremos, performance da
linguagem C e a produtividade de linguagem de script.

O uso de luajit, de arquiteturas de servidor de aplicação voltadas para realtime
e a produtividade de programação web semelhante Ruby on Rails fazem de Charon
não somente a mais revolucionária, mas a definitiva plataforma de desenvolvimento web !
